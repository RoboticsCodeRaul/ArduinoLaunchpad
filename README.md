# Arduino Launchpad

Combining a design idea with 3D Prints, WS2812 LEDs, tactile switches and an Arduino we will create a proper MIDI instrument.

### Components
* WS2812 LEDs
* 1x Arduino Nano
* 36x Tactile Switch
* 36x 1N4002 Diode
* 2x Perfboard
